# Agrotecx 🌾

**Agrotecx** is a multilingual, offline-capable farmer advisory tool that helps farmers make informed decisions about crop planning, pest management, and agricultural best practices. It features an intuitive interface with icons and supports regional languages for easy accessibility.

---

## Features

- **Offline & Multilingual Support** – Works without internet and supports regional languages.  
- **Crop Advisory** – Suggests optimal crops based on soil, climate, and season.  
- **Pest & Disease Management** – Provides insights for pest control and crop diseases.  
- **Farmer-Friendly UI** – Simple interface with understandable icons.  
- **Voice Assistant** – Interact using voice commands in multiple languages.
